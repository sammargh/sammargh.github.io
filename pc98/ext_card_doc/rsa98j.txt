RSA-98　（I・Oデータ）

Cバス高速RS-232C拡張ボード（調歩同期）．シリアルポートを2ch装備．各ch毎に送信／受信用
にそれぞれ2048Byte FIFO搭載（計8KB）．ボード上に8連ディップスイッチ1個と2本×7個ジャンパ
スイッチ1個あり．
シリアルコントロールLSIにTEXAS INSTRUMENTS社製TL16C550Cを採用し，最大230.4Kbpsの転送
速度で通信が可能（マスタクロック14.7456MHz）．1ch用データバスはDB0〜7，2ch用データバスは
DB8〜15．最大消費電力+5V, 500mA／+12V，30mA／-12V，35mA．使用環境は温度0〜40゜C／湿度
20〜80%（結露なきこと）．拡張I/Oユニットでの使用も可能．
RSA-98，RSA-98SB，RSA-98II/Sを複数枚使用してさらにRS-232Cポートを増設可能．添付のBIOSでは
最大32回線，またWindows3.1用COMドライバでは最大7回線のRS-232Cポートが制御可能（WindowsNT
3.5xおよびWindows95用のCOMドライバはRSA-98シリーズ1枚のみをサポート：WindowsNT3.5xと
Windows95ではRSA-98，RSA-98SB，RSA-98II/Sの複数枚使用は不可）．RSA-98には別売りのシリアル
ポートコネクタRSA-EXを用いてRSA-98SBを取り付け可能．
データの取りこぼしを防ぐため，フロー制御を設定して使用のこと．ハードフロー制御を有効に
るためには，必ずRTS/CTS，DTR/DSRの信号線の結線されているケーブルを使用のこと．

・8連ディップスイッチSW1：I/Oポートアドレスの上位8bitの設定
ディップは2進数のビットパタンに対応．
8個のディップを4個ずつ分けた場合，前半の4-3-2-1と後半の8-7-6-5のいずれにおいても，16進数
とディップのパターンの対応は以下の通り：
0■OFF-OFF-OFF-OFF
1■OFF-OFF-OFF-ON
2■OFF-OFF-ON-OFF
3■OFF-OFF-ON-ON
4■OFF-ON-OFF-OFF
5■OFF-ON-OFF-ON
6■OFF-ON-ON-OFF
7■OFF-ON-ON-ON
8■ON-OFF-OFF-OFF
9■ON-OFF-OFF-ON
A■ON-OFF-ON-OFF
B■ON-OFF-ON-ON
C■ON-ON-OFF-OFF
D■ON-ON-OFF-ON
E■ON-ON-ON-OFF
F■ON-ON-ON-ON
例）50hに設定する場合，5-6-7-8をOFF-ON-OFF-ON，また1-2-3-4をOFF-OFF-OFF-OFFにそれぞれ
設定する．なお出荷時設定は50h．
注1：PC-9821シリーズではI/Oポートアドレスの上位8ビットのx8h〜xFhが予約されているので，
　　 x0h〜x7hの範囲内で設定のこと．
注2：この記述はRSA-98シリーズ取扱説明書（1996年3月4日発行）に依っているが，同書の6〜7
     ページのSW1の図では，"OFF"に設定するにはディップスイッチを"ON"側にずらすような記述と
     なっている（RAS-98の取説には白黒いずれの矩形がディップスイッチを表すか書かれていない
　　 が，I・Oデータの他のボードの取説の記述から判断すると，本ボードでも黒い矩形がディップ
     スイッチを示しているものと思われる）．
例）6に設定する場合の取説での図
　　　８７６５
　　　４３２１
　　　□■■□
　　　■□□■
　　　　　　ON
I/Oポートアドレスの下位8bitは下記のようにハードウェアで固定．
(1)TL16C550Cコントロール用ポート
1ch：xxE0h，xxE2h，xxE4h，xxE6h，xxE8h，xxEAh，xxECh，xxEEh
2ch：xxE1h，xxE3h，xxE5h，xxE7h，xxE9h，xxEBh，xxEDh，xxEFh
(2)送受信FIFO用ポートアドレス
1ch：xxD0h，xxD2h，xxD4h，xxD6h
2ch：xxD1h，xxD3h，xxD5h，xxD7h
（要するに下位8bitはD0h〜D7h，E0h〜EFh固定）
※GA-98NBシリーズとの併用時，I/Oポートアドレスが出荷時設定（50h）のままでは重複し，正常
　動作しない．この場合，RSA-98のベースI/Oポートアドレスの上位8bitを40h〜5Fh以外（GA-98NBI/C
　との併用時はさらに30h〜32h以外）で，かつその他のボードと重複しないように設定のこと．

・2本×7個ジャンパスイッチJP1：割り込みレベルの設定
以下，割り込みレベル■ショートすべきピン　の順
INT 0■1 縦2本　（出荷時設定）
INT 1■2 縦2本
INT 2■3 縦2本
INT 3■4 縦2本
INT 41■5 縦2本
INT 5■6 縦2本
INT 6■7 縦2本
割り込み不使用■6の上1本と7の上1本（ジャンパプラグが横向きになる）
※1chと2chの割り込みレベルは同一．

COMドライバについて
(1)COMドライバ　for Windows95
・RSA-98のベースI/Oポートアドレスが出荷時設定の50hに固定されている．
・デバイスマネージャで「マルチファンクションアダプタ」と「拡張シリアルボード(RSA-98)」
　の順にクリック．次いで「プロパティ」で両ポートとも同じIRQに設定．
・セットアップ後，「拡張シリアルボード」(RSA-98)(COM2)および「拡張シリアルボード」
　(RSA-98)(COM3)が登録され，COM3に!マークがつくが動作に支障なし．
(2)COMドライバ for WindowsNT3.x
・RSA-98のベースI/Oポートアドレスが出荷時設定の50hに，また割り込みレベルがINT 0に固定
　されている．
・ドライバファイルのコピーが終了すると[WindowsNTセットアップ]の[マウス：]の箇所に
　[アイ・オーデータ高速シリアルボードRSA-98シリーズ]と表示されるがそのまま終了させる．
　再起動後[マウス：]の箇所を[PC-9800マウスを使用する]に戻すこと．
これらのドライバを使用するにあたり，RSA-98シリーズBIOSやRSA-98シリーズシリアルドライバを
組み込まないこと．
※MS-DOS，Windows3.1用のBIOS／ドライバについては略．
