In addition, some boards have unknown settings because there is no written description. Such boards show the state of the switch when I get it as a junk item.

※ If you do not know the switch settings of C bus SCSI board, PC-9821 and DOS / V software page- > System status display software (Analysis and verification tool) ---> SCSI_DSW (C bus SCSI board You can check the switch settings by yourself using a program that knows the settings of dip switches and jumper switches.This software may be effective even with a SCSI board for SCSI dedicated slots, but it has not been confirmed.
　Also, if the C bus SCSI board's ROM address and interrupt settings are not appropriate, the system can not be booted and SCSI board switch settings can not be read using SCSI_DSW. In that case, SCSI_DSW should be compatible with the FD loader PC-9821 and DOS / V software page- > PC-98's FD loader application tool (system setting / modification tool)-> SCSIDSW (a program to create floppy to read switch setting of C bus SCSI board) If you create a floppy disk in advance, you can use it to display the switch settings of that SCSI board.
